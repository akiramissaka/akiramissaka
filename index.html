Olá, meu nome é Akira Missaka e sou desenvolvedor web com foco em front-end. Tenho 29 anos e sou formado em Produção Multimídia (antes chamado comunicação para web). Tenho mais de 3 anos de experiência em desenvolvimento web. Possuo conhecimentos intermediários / avançados em HTML5 / CSS3 / Javascript / JQuery / Ajax / Layout Responsivo (Mobile First) / Photoshop (para web) / SEO / Bootstrap / ASP / SQL Server / MySQL, conhecimentos básicos em PHP e superficiais em ASP.Net / C#.

Trabalhei por mais de três anos como desenvolvedor front-end / back-end para a SGI Web (Grupo Remaza) que em seus sites internos priorizava a retrocompatibilidade com IE6 / IE7, e em seus sites e landing pages priorizava o bom funcionamento cross-browser e boas praticas SEO.

Também trabalhei por seis meses como desenvolvedor front-end para a iFabrica, recebendo arquivos psd de uma agência e transcrevendo-os para html / css / javascript. Nesses seis meses também trabalhei em paralelo no projeto de uma web app em HTML5 /CSS3 / jQuery / Bootstrap.

Abaixo segue meu portfolio com alguns dos trabalhos que realizei. Caso tenha interesse, estou disponível para contratação full time ou freelances. Entre em contato para conversarmos.
